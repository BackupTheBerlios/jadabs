Chat Swing-Application that uses Jadabs JXME-J2ME Messaging and Bluetooth Transport. This package contains a Chat Application that uses JXME-J2ME messaging and the JXME-BT transport for Bluetooth communication between the chat peers. The application is the counterpart of the JXME-Chat MIDlet for mobile phones. It can be used to chat from J2SE device with a CLDC/MIDP device runnign the MIDlet. It is a demo application for the newly implemented BT Layer for Jadabs-JXME.
