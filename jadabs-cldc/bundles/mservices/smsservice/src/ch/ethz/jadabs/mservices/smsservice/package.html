SMS Service that allows sending short messages using the Java Wireless Messaging API. This package contains an SMS Service that allows sending short messages encapsulated in JXME messages over the Java Wireless Messaging API (WMA) (JSR 120: Wireless Messaging API, JSR 205: Wireless Messaging API 2.0). The WMA is available on most of the J2ME smart phone devices. The package uses only WMA 1.0 features.
